树莓派远程控制与智能交互产品
 
完整产品功能文档（最终正式版·V1.0）
 
文档说明：整合全量需求，包含硬件、软件、云端、配网、通信、多用户管理、隐私加密、文件权限、公私空间等全部功能，无遗漏、无歧义、可直接用于开发、交付、立项。
 
 
 
一、产品概述
 
本产品是一套无外设、跨公网、端到端加密、强隐私保护的树莓派远程控制与智能交互系统，由树莓派硬件端、多平台软件客户端（手机APP/电脑客户端）、云端服务三部分组成。
 
产品核心能力：
 
- 蓝牙完成首次配网（WiFi + 会话链接 + 密钥分发）
- 公网环境通过云端会话链接实现无长连接远程控制
- 双交互入口：控制台（命令运维）+ 聊天框（自然语言智能交互）
- 单账号多设备管理、多用户群控共享设备
- 端到端E2EE加密，密钥仅蓝牙分发、不上公网
- 严格隐私保护：文件数据默认不出私域，支持三级隐私权限
- 多用户公私空间隔离：公共空间共享 + 个人隐私空间独立
 
产品定位：面向家庭、个人、小型团队的隐私优先型树莓派远程管理平台。
 
 
 
二、硬件端（树莓派）功能说明
 
2.1 硬件限制
 
- 无屏幕、无键盘、无鼠标、无任何外接外设
- 出厂无WiFi、无会话链接、无密钥信息
- 仅预装Linux系统、Docker、OpenClaw运行环境
- 所有配置仅支持命令行/软件端远程配置
 
2.2 硬件基础能力
 
- 支持WiFi连接公网/局域网
- 支持BLE蓝牙，仅用于首次初始化配网
- 可存储：WiFi信息、会话链接、公共密钥、个人用户密钥
- 定时轮询云端会话链接，拉取加密消息
- 消息解密、命令执行、文件存储均在本地完成
- 区分消息类型： cmd 命令 /  nl 自然语言
- 所有文件、数据仅本地存储，默认不上传云端
 
2.3 硬件工作模式
 
1. 出厂模式：开启蓝牙广播，等待配网
2. 配网模式：通过蓝牙接收WiFi、会话链接、密钥
3. 公网运行模式：关闭蓝牙，轮询云端，本地执行
4. 局域网直连模式：同一私网下支持高速文件同步
 
 
 
三、云端服务功能说明
 
3.1 核心定位
 
云端仅做透明中转，不存储明文、不解密、不解析业务数据、不建立长连接。
 
3.2 账号与会话管理
 
- 用户注册、登录、唯一身份标识
- 一台设备 → 一个永久唯一会话链接
- 支持单账号绑定多台树莓派
- 支持多用户绑定同一台设备
 
3.3 消息中转服务
 
- 接收加密消息并暂存
- 提供消息发送/拉取接口
- 不推送、不转发、不解析消息内容
- 离线消息缓存
 
3.4 权限管理服务
 
- 存储用户角色：管理员 / 普通成员
- 存储用户与设备、会话链接的绑定关系
- 不存储任何密钥、不参与加解密
 
 
 
四、软件客户端（手机APP/电脑客户端）功能说明
 
手机与电脑功能完全一致，仅做界面适配。
 
4.1 账号与设备管理
 
- 用户注册、登录
- 设备列表：展示账号下所有树莓派
- 每台设备独立控制台 + 独立聊天框
- 设备在线/离线状态展示
 
4.2 首次配网（仅管理员）
 
- 蓝牙搜索 → 连接树莓派
- 下发WiFi配置
- 申请云端生成会话链接
- 生成本地密钥（公共+个人）
- 通过蓝牙下发：会话链接 + 公共密钥 + 个人密钥
- 完成后断开蓝牙，进入公网模式
 
4.3 控制台模块（type=cmd）
 
- 发送Linux/Docker/OpenClaw/系统配置命令
- 消息加密后上传至会话链接
- 拉取并解密执行结果
- 命令历史、快捷操作、结果展示
 
4.4 聊天框模块（type=nl）
 
- 自然语言对话交互
- 群聊模式，显示发送人ID/昵称
- 消息加密传输
- OpenClaw+大模型执行结果展示
 
4.5 设备共享与多用户加入
 
- 管理员分享会话链接
- 新用户输入链接即可加入
- 无需蓝牙、无需配网
- 新用户自动获取公共密钥，生成个人密钥
 
4.6 文件管理与隐私权限
 
- 局域网直连：完整文件读写、预览、下载、同步
- 远程公网：按三级隐私权限控制访问
- 公私空间切换：公共空间 / 个人隐私空间
- 文件本地解密，云端不可见
 
4.7 加密与密钥管理
 
- 本地生成密钥
- 密钥仅蓝牙传输
- 消息自动加解密
- 密钥永不触网、不上传
 
 
 
五、核心通信架构（无长连接·会话链接机制）
 
5.1 通信规则
 
- 无任何长连接
- 所有消息通过云端会话链接收发
- 模式：发送 → 存储 → 拉取
- 跨公网、跨局域网可用
- 无需端口映射、公网IP
 
5.2 标准消息格式（加密后上传）
 
plaintext
  
session_id        会话ID
from_user_id      发送者ID
from_user_role    角色
msg_type          cmd/nl
content           加密后的内容
timestamp         时间
 
 
5.3 端到端加密（E2EE）流程
 
1. 客户端本地生成密钥
2. 蓝牙传给树莓派
3. 发送方：明文 → 加密 → 上传云端
4. 接收方：拉取密文 → 本地解密 → 展示
5. 云端仅存储密文，无密钥无法解密
 
 
 
六、数据隐私 & 文件回传同步机制
 
6.1 核心隐私原则
 
- 数据默认绝对不出公网
- 除非用户明确授权，否则文件不经云端
- 公网无法获取真实文件内容
- 平台无法查看用户任何明文数据
 
6.2 文件访问权限分级（用户自主选择）
 
等级 0：最强隐私（默认）
 
- 远程不可查看任何文件、目录、结构
- 0数据云端传输，最高安全
 
等级 1：仅目录可见（平衡档）
 
- 可查看目录、文件名、大小、时间
- 禁止查看内容、预览、下载
- 仅少量非敏感目录信息走云端
 
等级 2：有限预览（便捷档）
 
- 可查看缩略图、文件头、少量文字
- 禁止查看全文、禁止下载源文件
- 仅极低敏感内容中转
 
6.3 文件同步规则
 
- 所有文件仅存储在树莓派本地
- 同一局域网：直连高速同步
- 公网远程：任何等级都不支持完整文件传输
- 完整文件只能在局域网同步到用户电脑后查看
 
 
 
七、多用户 + 公私空间 + 加密权限系统
 
7.1 两套密钥体系（核心安全架构）
 
① 公共空间密钥（共享）
 
- 生成：首次配网生成
- 分发：蓝牙本地传给所有授权用户
- 用途：公共文件夹、共享文件、群聊消息
- 所有成员可解密
 
② 个人隐私空间密钥（独属）
 
- 生成：用户设备本地生成
- 分发：仅蓝牙传给树莓派，不上公网
- 用途：个人文件、私人数据
- 仅用户自己可解密，他人/管理员/平台均不可见
 
7.2 空间权限定义
 
1. 公共空间
所有授权用户可读写、查看、共享
2. 个人隐私空间
仅本人可见、可操作，绝对私密
3. 管理员
可管理用户、分配空间、配置权限
无法查看他人隐私空间内容
 
7.3 数据安全规则
 
- 所有文件先加密，再存储
- 树莓派只存密文
- 公网只能看到加密数据
- 局域网内才能解密、查看、同步
 
 
 
八、完整流程说明
 
8.1 首次初始化配网流程（唯一一次蓝牙）
 
1. 树莓派上电 → 开启蓝牙
2. 管理员蓝牙连接树莓派
3. 下发WiFi配置 → 树莓派连公网
4. 云端生成唯一会话链接
5. 客户端生成：公共密钥 + 个人密钥
6. 蓝牙下发：会话链接 + 双密钥
7. 树莓派保存配置 → 关闭蓝牙
8. 正式进入公网加密运行模式
 
8.2 公网远程消息流程（加密）
 
1. 用户发送命令/消息
2. 本地加密
3. 上传云端会话链接
4. 树莓派轮询拉取密文
5. 本地解密 → 执行
6. 结果加密 → 回传云端
7. 用户拉取 → 解密 → 展示
 
8.3 多用户加入流程（无需蓝牙）
 
1. 管理员分享会话链接
2. 新用户绑定设备
3. 自动获取公共密钥
4. 生成本人隐私密钥
5. 通过蓝牙/局域网传给树莓派
6. 立即进入群控群聊模式
 
 
 
九、权限体系
 
9.1 角色权限
 
管理员（创建者）
 
- 蓝牙配网
- 修改WiFi
- 生成/分享会话链接
- 管理用户
- 配置文件隐私等级
- 使用公共空间 + 个人空间
 
普通成员
 
- 远程控制、发送命令/聊天
- 使用公共空间 + 个人空间
- 无法配网、无法分享设备
- 无法查看他人隐私空间
 
9.2 文件权限
 
- 公共空间：全员可访问
- 个人空间：仅本人可访问
- 远程访问：按三级隐私权限限制
- 局域网：完全开放
 
 
 
十、产品核心安全优势（可对外宣传）
 
1. 真正端到端加密（E2EE）
2. 密钥仅蓝牙分发，永不触网
3. 平台无法解密、无法查看用户任何明文数据
4. 数据默认不出私域，隐私架构级保障
5. 多用户隔离，公私空间分离
6. 无长连接、无主动推送，攻击面极低
7. 远程仅开放有限信息，完整数据局域网同步
